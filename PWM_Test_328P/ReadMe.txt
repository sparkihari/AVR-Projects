Pulse Width Modulation (PWM) Test

 Hardware Required:

 1, Arduino Uno / Nano Board
 

 PWM Pins:

-> PWM0 - PD6/OC0A/D6               // 244 Hz
-> PWM1 - PD5/OC0B/D7
-> PWM2 - PB3/OC2A/D11
-> PWM3 - PD3/OC2B/D3

-> PWM4 - PB1/OC1A/D9               // 50 Hz
-> PWM5 - PB2/OC1B/D10

Note: the timers (0 and 2) are using the Fast PWM mode
